# Memory Palace — پروژهٔ حرفه‌ای (MemorTPalace)

یک پروژهٔ دسکتاپ قدرتمند برای تقویت حافظه و ایجاد "قصرِ حافظه" شخصی. این پروژه با رویکرد شیءگرایی (OOP) طراحی شده، از `sqlite` برای ذخیره‌سازی دیتابیس محلی استفاده می‌کند، رابط کاربری با `tkinter` ساخته شده و از الگوی تولیدکننده (generators) در منطق برنامه بهره می‌برد.

**ویژگی‌ها**
- **ذخیره‌سازی امن و محلی:** استفاده از دیتابیس `sqlite` برای نگهداری کارت‌ها، دسته‌ها و آمار.
- **معماری OOP:** ساختار ماژولار و قابل توسعه با کلاس‌های جداگانه برای مدل، ویو و کنترل.
- **رابط کاربری گرافیکی:** ساخته‌شده با `tkinter` برای تجربهٔ ساده و قابل‌فهم.
- **تولید هوشمند محتوا:** استفاده از `generator`ها برای ایجاد توالی تمرین و بازگشتِ پراکنده (spaced repetition) به‌صورت قابل‌توسعه.
- **قابل توسعه:** آماده برای افزودن الگوریتم‌های یادگیری ماشینی یا همگام‌سازی ابری.

**فناوری‌ها**
- Python 3.8+
- OOP (کلاس‌ها و الگوهای طراحی)
- `sqlite3` (دیتابیس محلی)
- `tkinter` (رابط کاربری)
- Generators و الگوریتم‌های واکشی و صف‌بندی

**نصب و اجرا**
1. کد را کلون کنید یا فولدر پروژه را دانلود کنید.
2. (اختیاری) یک محیط مجازی بسازید و فعال کنید:

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1  # PowerShell
```

3. (اگر نیاز به بسته‌های خارجی بود) آن‌ها را نصب کنید. این پروژه با کتابخانهٔ استاندارد پایتون کار می‌کند.
4. برنامه را اجرا کنید:

```bash
python main.py
```

**ساختار پیشنهادی‌ٔ پروژه**
- `main.py` — نقطهٔ ورود برنامه و تنظیمات اولیه.
- `assets/` — تصاویر و منابع گرافیکی.
- ماژول‌های `models`, `views`, `controllers` — تقسیم‌بندی براساس OOP.

**نحوهٔ استفاده (راهنمای سریع)**
- برنامه را اجرا کنید؛ پنجرهٔ اصلی با گزینهٔ ایجاد کارت جدید، لیست کارت‌ها و جلسه‌های تمرینی نمایش داده می‌شود.
- کارت‌ها در `sqlite` ذخیره می‌شوند و می‌توانید آنها را ویرایش یا حذف کنید.
- سیستم تولیدِ جلسه‌ها با generator قابل شخصی‌سازی است (مثلاً افزودن الگوریتم‌های SRS).

**نحوهٔ توسعه و افزودن قابلیت‌ها**
- افزودن لاگین/همگام‌سازی ابری با یک سرویس REST.
- پیاده‌سازی الگوریتم‌های SRS پیشرفته و وارد کردن داده‌ها از فایل‌های CSV.
- تبدیل رابط به `tkinter` → `ttk` یا مهاجرت به فریم‌ورک‌های مدرن‌تر (Qt, Electron) در صورت نیاز.
